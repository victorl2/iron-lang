   - iron_arena_create: malloc(capacity), set used=0
   - iron_arena_alloc: align-up via (used + align - 1) & ~(align - 1), grow via realloc if needed (double capacity), return base + aligned
   - iron_arena_strdup: alloc len+1 bytes with align=1, memcpy, null-terminate
   - iron_arena_free: free(base), zero all fields

   - void iron_strbuf_append(Iron_StrBuf *sb, const char *str, size_t len);
   - void iron_strbuf_appendf(Iron_StrBuf *sb, const char *fmt, ...) with __attribute__((format(printf, 2, 3)));
   - void iron_strbuf_append_char(Iron_StrBuf *sb, char c);
   - const char *iron_strbuf_get(Iron_StrBuf *sb);
   - void iron_strbuf_free(Iron_StrBuf *sb);
   - void iron_strbuf_reset(Iron_StrBuf *sb);

6. Create src/util/strbuf.c implementing:

   - iron_strbuf_get: return data (already null-terminated)
   - iron_strbuf_free: free(data), zero fields
   - iron_strbuf_reset: set len=0, data[0]='\0'

   - Include guard: #ifndef IRON_DIAGNOSTICS_H / #define IRON_DIAGNOSTICS_H

   - Iron_Span struct: const char *filename (interned), uint32_t line (1-indexed), uint32_t col (1-indexed, byte column), uint32_t end_line, uint32_t end_col
   - Helper: Iron_Span iron_span_make(const char *filename, uint32_t line, uint32_t col, uint32_t end_line, uint32_t end_col);
   - Helper: Iron_Span iron_span_merge(Iron_Span start, Iron_Span end); — creates span from start of first to end of second
   - Iron_DiagLevel enum: IRON_DIAG_ERROR, IRON_DIAG_WARNING, IRON_DIAG_NOTE
   - Iron_Diagnostic struct: Iron_DiagLevel level, int code (E-code number), Iron_Span span, const char *message (arena-allocated), const char *suggestion (NULL if none)
   - Iron_DiagList struct: Iron_Diagnostic *items (stb_ds dynamic array), int count, int error_count, int warning_count
   - Iron_DiagList iron_diaglist_create(void);
   - void iron_diag_emit(Iron_DiagList *list, Iron_Arena *arena, Iron_DiagLevel level, int code, Iron_Span span, const char *message, const char *suggestion);
   - void iron_diag_print(const Iron_Diagnostic *d, const char *source_text);
   - void iron_diag_print_all(const Iron_DiagList *list, const char *source_text);
   - void iron_diaglist_free(Iron_DiagList *list);
   - Error code constants:
     #define IRON_ERR_UNTERMINATED_STRING  1
     #define IRON_ERR_INVALID_CHAR         2
     #define IRON_ERR_INVALID_NUMBER       3
     #define IRON_ERR_UNEXPECTED_TOKEN    101
     #define IRON_ERR_EXPECTED_EXPR       102
     #define IRON_ERR_EXPECTED_RBRACE     103
     #define IRON_ERR_EXPECTED_RPAREN     104
     #define IRON_ERR_EXPECTED_COLON      105
     #define IRON_ERR_EXPECTED_ARROW      106

2. Create src/diagnostics/diagnostics.c implementing:
   - iron_span_make: fill and return Iron_Span struct
   - iron_span_merge: take start line/col from first span, end line/col from second span, filename from first
   - iron_diaglist_create: zero-init struct, items = NULL (stb_ds empty)
   - iron_diag_emit: arena-strdup the message string, arena-strdup suggestion if non-NULL, fill Iron_Diagnostic, arrput into list->items, increment count and error_count/warning_count
   - iron_diag_print: Rust-style format with 3-line context window:
     Format: "error[E%04d]: %s\n  --> %s:%u:%u\n" then source lines with | gutter, ^ pointing to error column, "= help: %s" if suggestion is non-NULL.
     Use isatty(STDERR_FILENO) to decide whether to emit ANSI color codes (\033[1;31m for error, \033[1;33m for warning, \033[0m reset).
     Extract source lines by scanning source_text for newlines.
   - iron_diag_print_all: iterate list->items, call iron_diag_print for each
   - iron_diaglist_free: stb_ds arrfree(list->items), zero fields
