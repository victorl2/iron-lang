       - `Iron_RcControl` struct: `atomic_int strong_count`, `atomic_int weak_count`, `void (*destructor)(void *value)`
       - `Iron_Rc` struct: `Iron_RcControl *ctrl` + `void *value`
       - `Iron_Weak` struct: `Iron_RcControl *ctrl`
       - `Iron_Rc iron_rc_create(void *value, size_t size, void (*destructor)(void *))` — allocate control block + copy value
       - `void iron_rc_retain(Iron_Rc *rc)` — atomic increment strong count
       - `void iron_rc_release(Iron_Rc *rc)` — atomic decrement; free on zero
       - `Iron_Weak iron_rc_downgrade(const Iron_Rc *rc)` — create weak reference
       - `Iron_Rc iron_weak_upgrade(const Iron_Weak *weak)` — try to get strong ref (NULL ctrl if dead)
    6. Iron_Error type: `typedef struct { int code; const char *message; } Iron_Error;`
       - `Iron_Error iron_error_none(void)` — returns {0, NULL}
       - `Iron_Error iron_error_new(int code, const char *msg)` — creates error
       - `bool iron_error_is_ok(Iron_Error e)` — returns e.code == 0
    7. Built-in function declarations:
       - `void Iron_print(Iron_String s)`
       - `void Iron_println(Iron_String s)`
       - `int64_t Iron_len(Iron_String s)`
       - `int64_t Iron_min(int64_t a, int64_t b)`

       - `void iron_runtime_shutdown(void)` — cleanup

    **iron_string.c** — implementation:
    1. Static intern table: `static struct { char *key; Iron_String value; } *s_intern_table = NULL;`
    2. Thread-safe intern lock: `static pthread_mutex_t s_intern_lock = PTHREAD_MUTEX_INITIALIZER;`
    3. `iron_string_from_cstr`: if byte_len <= IRON_STRING_SSO_MAX, store inline in sso.data, set sso.len; else heap-allocate, set heap.data/byte_length/codepoint_count/flags
    4. `iron_string_from_literal`: call from_cstr then intern
    5. `iron_string_cstr`: return sso.data if inline, else heap.data
    6. UTF-8 codepoint counting: standard 4-line advance loop (check leading byte bits: 0xxxxxxx=1, 110xxxxx=2, 1110xxxx=3, 11110xxx=4)
    7. `iron_string_equals`: compare byte lengths first, then memcmp
    8. `iron_string_concat`: compute total len, create new string (SSO or heap), memcpy both parts
    9. `iron_string_intern`: lock s_intern_lock, check shgeti, if found return existing, else shput new entry, unlock
    10. `iron_runtime_init`: initialize intern table (sh_new_strdup); global pool init is deferred to Plan 02
    11. `iron_runtime_shutdown`: free intern table entries, shfree
    12. Include `<pthread.h>` for intern lock, `<string.h>`, `<stdlib.h>`, `<stdio.h>`
    13. `#define STB_DS_IMPLEMENTATION` must NOT be here — it's already in src/util/stb_ds_impl.c. Just `#include "vendor/stb_ds.h"`.

    **iron_rc.c** — implementation:
    1. `iron_rc_create`: malloc(sizeof(Iron_RcControl) + size), init strong=1, weak=0, memcpy value after control block, set rc.ctrl and rc.value
    2. `iron_rc_retain`: `atomic_fetch_add(&rc->ctrl->strong_count, 1)`
    3. `iron_rc_release`: `atomic_fetch_sub(&rc->ctrl->strong_count, 1)`; if old value was 1: call destructor, then if weak_count == 0 free ctrl
    4. `iron_rc_downgrade`: `atomic_fetch_add(&ctrl->weak_count, 1)`, return Iron_Weak{ctrl}
    5. `iron_weak_upgrade`: atomically check strong > 0, if so increment strong and return Iron_Rc{ctrl, value}; else return {NULL, NULL}
    6. Weak upgrade needs CAS loop: `atomic_compare_exchange_weak` on strong_count

    **iron_builtins.c** — implementation:
    1. `Iron_print(Iron_String s)`: get cstr, printf("%s", cstr) — no newline
    2. `Iron_println(Iron_String s)`: get cstr, printf("%s\n", cstr)
    3. `Iron_len(Iron_String s)`: return iron_string_codepoint_count(&s) cast to int64_t
    4. `Iron_min/Iron_max/Iron_clamp/Iron_abs`: simple integer arithmetic
    5. `Iron_assert(bool cond, Iron_String msg)`: if (!cond) { fprintf(stderr, "assertion failed: %s\n", iron_string_cstr(&msg)); abort(); }
