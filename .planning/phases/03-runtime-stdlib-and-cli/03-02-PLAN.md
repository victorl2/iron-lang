    **iron_threads.c** — implementation:

    1. **Pool worker function** `static void *pool_worker(void *arg)`:
       - Loop: lock pool->lock, while queue empty and !shutdown: cond_wait(work_ready, lock)
       - If shutdown && queue empty: unlock, return NULL
       - Dequeue item from circular buffer (head advances mod capacity)
       - Unlock, execute fn(arg), lock, decrement pending, if pending==0 cond_broadcast(work_done), unlock

    2. **Iron_pool_create(name, thread_count)**:
       - malloc pool struct, set name, thread_count
       - Initial queue capacity: 256
       - malloc queue array, init head=tail=0, pending=0, shutdown=false
       - pthread_mutex_init, pthread_cond_init x2
       - Create thread_count threads via pthread_create, each running pool_worker

    3. **Iron_pool_submit(pool, fn, arg)**:
       - Lock, if queue full (count == capacity): realloc queue to 2x capacity (copy circular buffer to linear first)
       - Enqueue at tail, increment count and pending, cond_signal(work_ready), unlock

    4. **Iron_pool_barrier(pool)**:
       - Lock, while pending > 0: cond_wait(work_done, lock), unlock

    5. **Iron_pool_destroy(pool)**:
       - Lock, set shutdown=true, cond_broadcast(work_ready), unlock
       - pthread_join all threads, free threads array, free queue, destroy mutex/condvars, free pool

    6. **Iron_pool_thread_count**: return pool->thread_count

    7. **Iron_handle_create(fn, arg)**:
       - malloc Iron_Handle, init lock+cond, done=false, panic_msg=NULL
       - Create a wrapper that runs fn(arg), sets done=true, signals cond
       - pthread_create with the wrapper

    8. **Iron_handle_wait(handle)**:
       - Lock, while !done: cond_wait, unlock
       - If panic_msg != NULL: fprintf(stderr, "panic in spawned task: %s\n", panic_msg); abort()
       - pthread_join the thread

    9. **Channel implementation**:
       - create: capacity = max(capacity, 1) (unbuffered = capacity 1 per research)
       - send: lock, while count==capacity && !closed: cond_wait(not_full, lock); if closed: unlock+return; ring[tail] = item; tail = (tail+1)%capacity; count++; cond_signal(not_empty); unlock
       - recv: lock, while count==0 && !closed: cond_wait(not_empty, lock); if count==0 && closed: unlock+return NULL; item = ring[head]; head = (head+1)%capacity; count--; cond_signal(not_full); unlock; return item
       - try_recv: lock, if count==0: unlock+return false; else dequeue, return true
       - close: lock, closed=true, broadcast both condvars, unlock
       - destroy: free ring, destroy mutex/condvars, free channel

    10. **Mutex implementation**:
        - create: malloc struct, malloc value copy of size, memcpy initial_value, pthread_mutex_init
        - lock: pthread_mutex_lock, return value pointer
        - unlock: pthread_mutex_unlock
        - destroy: free value, pthread_mutex_destroy, free struct

    11. **Lock/CondVar**: thin wrappers around pthread_mutex/cond functions

    12. **Global pool init**: In iron_runtime_init() (in iron_string.c or new iron_runtime.c), set `Iron_global_pool = Iron_pool_create("global", cpu_count - 1)` where cpu_count is from `sysconf(_SC_NPROCESSORS_ONLN)` (or 4 as fallback). In iron_runtime_shutdown(): `Iron_pool_destroy(Iron_global_pool)`.

    Note: The `iron_runtime_init/shutdown` functions are defined in Plan 01's iron_string.c. Either add the pool init there, or create a small iron_runtime.c that calls both string init and pool init. Prefer: add pool init/shutdown to the existing iron_runtime_init/shutdown stubs.
