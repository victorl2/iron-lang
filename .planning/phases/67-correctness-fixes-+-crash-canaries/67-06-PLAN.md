```c
/* Before */
q->items = realloc(q->items, new_cap * sizeof(void *));

void **new_items = realloc(q->items, new_cap * sizeof(void *));
if (!new_items) iron_oom_abort("iron_threads.c:pool_queue_grow");
q->items = new_items;
```

For iron_threads.c IRON_THREAD_CREATE return-value checks (AUDIT-03 §32):
```c
/* Before */
IRON_THREAD_CREATE(&handle->thread, worker_fn, handle);
```
