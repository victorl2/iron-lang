**src/analyzer/types.c:**
Implement all the above. Intern primitives in a static array `s_primitives[IRON_TYPE_ERROR+1]` initialized by `iron_types_init`. `iron_type_make_primitive` returns `&s_primitives[kind]` for primitive kinds. Structural equality per RESEARCH.md code example. `iron_type_to_string` returns strings like "Int", "Float", "Bool", "String", "T?" for nullable, "[T; N]" for arrays, "func(...) -> R" for function types.

**src/analyzer/scope.h:**
Define `Iron_ScopeKind` enum: `IRON_SCOPE_GLOBAL, IRON_SCOPE_MODULE, IRON_SCOPE_FUNCTION, IRON_SCOPE_BLOCK`.
Define `Iron_SymbolKind` enum: `IRON_SYM_VARIABLE, IRON_SYM_FUNCTION, IRON_SYM_METHOD, IRON_SYM_TYPE, IRON_SYM_ENUM, IRON_SYM_ENUM_VARIANT, IRON_SYM_INTERFACE, IRON_SYM_PARAM, IRON_SYM_FIELD`.
Define `Iron_Symbol` struct with:
- `const char *name`
- `Iron_SymbolKind sym_kind`
- `Iron_Type *type` (resolved type; set by type checker)
- `Iron_Node *decl_node` (back-pointer to AST declaration)
- `Iron_Span span` (declaration location)
- `bool is_mutable` (true for var, false for val)
- `bool is_private`

Define stb_ds symbol entry struct:
```c
typedef struct { char *key; Iron_Symbol *value; } Iron_SymbolEntry;
```

Define `Iron_Scope` struct:
- `struct Iron_Scope *parent`
- `Iron_SymbolEntry *symbols` (stb_ds hash map)
- `Iron_ScopeKind kind`
- `const char *owner_name` (for debugging: function name, type name, or NULL)

Declare functions:
- `Iron_Scope *iron_scope_create(Iron_Arena *a, Iron_Scope *parent, Iron_ScopeKind kind)`
- `Iron_Symbol *iron_scope_lookup(Iron_Scope *s, const char *name)` — walks parent chain
- `Iron_Symbol *iron_scope_lookup_local(Iron_Scope *s, const char *name)` — current scope only
- `bool iron_scope_define(Iron_Scope *s, Iron_Arena *a, Iron_Symbol *sym)` — returns false if duplicate
- `Iron_Symbol *iron_symbol_create(Iron_Arena *a, const char *name, Iron_SymbolKind kind, Iron_Node *decl, Iron_Span span)`

**src/analyzer/scope.c:**
Implement all the above. Use `sh_new_strdup(s->symbols)` in `iron_scope_create`. Use `shgeti` to check for duplicates before `shput` in `iron_scope_define`. Walk parent chain in `iron_scope_lookup`. `iron_scope_lookup_local` only checks current scope.
