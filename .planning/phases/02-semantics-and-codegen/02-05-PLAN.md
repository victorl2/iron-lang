**src/codegen/codegen.h:**
```c
#ifndef IRON_CODEGEN_H
#define IRON_CODEGEN_H

#include "parser/ast.h"
#include "analyzer/scope.h"
#include "analyzer/types.h"
#include "diagnostics/diagnostics.h"
#include "util/arena.h"
#include "util/strbuf.h"

typedef struct {
    Iron_Arena   *arena;
    Iron_DiagList *diags;
    Iron_Scope   *global_scope;

    /* Output buffers — one per emission section */
    Iron_StrBuf   includes;        /* #include directives */
    Iron_StrBuf   forward_decls;   /* typedef struct X X; */
    Iron_StrBuf   struct_bodies;   /* struct X { ... }; */
    Iron_StrBuf   enum_defs;       /* typedef enum { ... } X; */
    Iron_StrBuf   prototypes;      /* void X_foo(X* self, ...); */
    Iron_StrBuf   implementations; /* void X_foo(X* self, ...) { ... } */
    Iron_StrBuf   main_wrapper;    /* int main() { ... } */

    /* Defer stack — dynamic array of dynamic arrays */
    Iron_Node  ***defer_stacks;    /* stb_ds: array of (array of Iron_Node*) */
    int           defer_depth;
    int           function_scope_depth;

    /* Type tags for inheritance */
    int           next_type_tag;

    /* Indentation */
    int           indent;

    /* Optional struct registry — track emitted Iron_Optional_T names */
    char        **emitted_optionals; /* stb_ds string array */
} Iron_Codegen;

/* Generate C code for the given analyzed program.
 * Returns the complete C source as a string (arena-allocated).
 * Returns NULL if the program has semantic errors.
 */
const char *iron_codegen(Iron_Program *program, Iron_Scope *global_scope,
                         Iron_Arena *arena, Iron_DiagList *diags);

/* --- Internal functions used across gen_*.c files --- */

/* Map Iron type to C type string. E.g., IRON_TYPE_INT -> "int64_t" */
const char *iron_type_to_c(const Iron_Type *t, Iron_Codegen *ctx);

/* Mangle an Iron name to C: "Player" -> "Iron_Player" */
const char *iron_mangle_name(const char *name, Iron_Arena *arena);

/* Mangle a method name: ("Player", "update") -> "Iron_Player_update" */
const char *iron_mangle_method(const char *type_name, const char *method_name, Iron_Arena *arena);

/* Emit indentation to the given buffer */
void codegen_indent(Iron_StrBuf *sb, int level);

/* Emit expression to buffer */
void emit_expr(Iron_StrBuf *sb, Iron_Node *node, Iron_Codegen *ctx);

/* Emit statement to buffer */
void emit_stmt(Iron_StrBuf *sb, Iron_Node *node, Iron_Codegen *ctx);

/* Emit block body (statements within braces) */
void emit_block(Iron_StrBuf *sb, Iron_Block *block, Iron_Codegen *ctx);

/* Emit defer drain from current depth down to target_depth */
void emit_defers(Iron_StrBuf *sb, Iron_Codegen *ctx, int target_depth);

/* Ensure Iron_Optional_T struct is emitted for the given type */
void ensure_optional_type(Iron_Codegen *ctx, const Iron_Type *inner);

#endif
```

**src/codegen/codegen.c:**
Implement `iron_codegen()` orchestrator:

1. Initialize `Iron_Codegen` context. Create all Iron_StrBuf sections with `iron_strbuf_create(4096)`.
2. Emit includes section:
```c
iron_strbuf_appendf(&ctx.includes, "#include <stdint.h>\n");
iron_strbuf_appendf(&ctx.includes, "#include <stdbool.h>\n");
iron_strbuf_appendf(&ctx.includes, "#include <stdlib.h>\n");
iron_strbuf_appendf(&ctx.includes, "#include <string.h>\n");
iron_strbuf_appendf(&ctx.includes, "#include <stdio.h>\n");
// Note: iron_runtime.h will be added in Phase 3
```

3. **Forward declarations pass:** Walk program->decls for ObjectDecl and emit:
   `typedef struct Iron_Player Iron_Player;\n`

4. **Topological sort** of struct definitions: Build dependency graph from object fields. Value-type field that references another object = dependency. Pointer/rc/nullable = no dependency (uses forward decl). DFS with gray/black coloring to detect cycles (E0223 IRON_ERR_CIRCULAR_TYPE). Emit struct bodies in sorted order.

5. **Struct bodies:** For each sorted ObjectDecl:
   - If has extends_name: embed parent as `_base` first field.
   - Else if participates in `is` checks (any child extends it): add `int32_t iron_type_tag;` as first field.
   - Emit each field with C type name.

6. **Enum definitions:** For each EnumDecl: `typedef enum { Iron_EnumName_VARIANT1, ... } Iron_EnumName;`

7. **Function prototypes:** For each FuncDecl/MethodDecl: emit the C function signature with `;`

8. **Function implementations:** Call emit_stmt for each FuncDecl/MethodDecl body.

9. **Main wrapper:**
```c
iron_strbuf_appendf(&ctx.main_wrapper, "int main(int argc, char** argv) {\n");
iron_strbuf_appendf(&ctx.main_wrapper, "    Iron_main();\n");
iron_strbuf_appendf(&ctx.main_wrapper, "    return 0;\n");
iron_strbuf_appendf(&ctx.main_wrapper, "}\n");
```

10. Concatenate all sections in order into final output.

Implement `iron_mangle_name`: prepend "Iron_" to name. Use `iron_arena_strdup` to allocate.

Implement `iron_mangle_method`: format as "Iron_{type}_{method}".

Implement `codegen_indent`: emit `indent * 4` spaces.

**src/codegen/gen_exprs.c:**
Implement `emit_expr()` — recursive switch on node kind:

- `IRON_NODE_INT_LIT`: `iron_strbuf_appendf(sb, "%s", lit->value)`
- `IRON_NODE_FLOAT_LIT`: `iron_strbuf_appendf(sb, "%s", lit->value)`
- `IRON_NODE_BOOL_LIT`: `iron_strbuf_appendf(sb, "%s", lit->value ? "true" : "false")`
- `IRON_NODE_STRING_LIT`: `iron_strbuf_appendf(sb, "\"%s\"", lit->value)` (Phase 2 stub: raw C string; Iron_String in Phase 3)
- `IRON_NODE_NULL_LIT`: emit `{ .has_value = false }` (for nullable context) or `NULL`
- `IRON_NODE_IDENT`: emit mangled name. If resolved_sym is a variable: emit the C name directly. If it's `self`: emit `self`.
- `IRON_NODE_BINARY`: `emit_expr(left); op_str; emit_expr(right)` where op_str maps Iron_OpKind to C operator. Wrap in parens for precedence: `(left op right)`.
  - `IRON_TOK_AND` -> `&&`, `IRON_TOK_OR` -> `||`, `IRON_TOK_NOT` -> `!`
  - `IRON_TOK_EQUALS` -> `==`, `IRON_TOK_NOT_EQUALS` -> `!=`
  - Arithmetic: `+`, `-`, `*`, `/`, `%`
- `IRON_NODE_UNARY`: for `IRON_TOK_MINUS` -> `(-expr)`, for `IRON_TOK_NOT` -> `(!expr)`
- `IRON_NODE_CALL`: if callee is a simple ident, emit `Iron_callee_name(args...)`. Handle printf/println as `printf` stub for Phase 2.
- `IRON_NODE_METHOD_CALL`: emit `Iron_Type_method(&obj, args...)`. The object is passed as first argument (pointer to self).
- `IRON_NODE_FIELD_ACCESS`: emit `obj.field` or `obj->field` depending on context (pointer vs value). For method self: `self->field`.
- `IRON_NODE_CONSTRUCT`: emit `(Iron_Type){ .field1 = arg1, .field2 = arg2, ... }` using field names from the object declaration. If parent has inheritance: include `.iron_type_tag = IRON_TAG_Type`.
- `IRON_NODE_INDEX`: emit `obj[idx]`
- `IRON_NODE_HEAP`: emit `malloc(sizeof(T))` + field initialization
- `IRON_NODE_IS`: emit `((Iron_BaseType*)ptr)->iron_type_tag == IRON_TAG_TypeName`

**src/codegen/gen_stmts.c:**
Implement `emit_stmt()` — recursive switch on node kind:

- `IRON_NODE_VAL_DECL`: emit `const <c_type> <mangled_name> = <init_expr>;`
- `IRON_NODE_VAR_DECL`: emit `<c_type> <mangled_name> = <init_expr>;`. If type is nullable: emit `Iron_Optional_T name = { .value = ..., .has_value = true/false };`
- `IRON_NODE_ASSIGN`: emit `<target> = <value>;`. For compound: `<target> += <value>;` etc.
- `IRON_NODE_RETURN`: **CRITICAL: emit defers first.** Call `emit_defers(sb, ctx, ctx->function_scope_depth)` then emit `return <value>;`
- `IRON_NODE_IF`: emit `if (<cond>) {` + body + `}`. Handle elif as `else if`. Handle else.
- `IRON_NODE_WHILE`: emit `while (<cond>) {` + body + `}`
- `IRON_NODE_FOR`: if not parallel: emit `for (int64_t <var> = 0; <var> < <range_end>; <var>++) {` + body + `}` (for range-based). For-in over collections: stub for Phase 3 runtime.
- `IRON_NODE_MATCH`: emit as if/else chain comparing subject to each case pattern.
- `IRON_NODE_DEFER`: push the defer expr onto `ctx->defer_stacks[ctx->defer_depth - 1]`. Do NOT emit anything here.
- `IRON_NODE_FREE`: emit `free(<expr>);`
- `IRON_NODE_LEAK`: emit nothing (leak is a semantic annotation, no C code)
- `IRON_NODE_BLOCK`: push defer level, emit each statement, drain defers at end, pop defer level.

Implement `emit_block()`: push block scope/defer, emit statements, drain block-level defers, pop.

Implement `emit_defers()` per RESEARCH.md Pattern 6:
```c
void emit_defers(Iron_StrBuf *sb, Iron_Codegen *ctx, int target_depth) {
    for (int d = ctx->defer_depth - 1; d >= target_depth; d--) {
        int n = arrlen(ctx->defer_stacks[d]);
        for (int i = n - 1; i >= 0; i--) {
            codegen_indent(sb, ctx->indent);
            emit_expr(sb, ctx->defer_stacks[d][i], ctx);
            iron_strbuf_appendf(sb, ";\n");
        }
    }
}
```
