1. Create `src/ir/lower_stmts.c` implementing the full `lower_stmt()` function. This replaces the stub in lower.c.

```c
/* lower_stmts.c — Statement lowering from AST to IR.
 * Mirrors gen_stmts.c but produces IronIR instructions. */

#include "ir/lower_internal.h"
#include "lexer/lexer.h"
#include <string.h>
```

The `lower_stmt(ctx, node)` function is a switch on `node->kind`. Note: the stub lower_stmt in lower.c (from Plan 08-01) already handles VAL_DECL, VAR_DECL, and ASSIGN. Move that logic into this file and remove the stub from lower.c.

**IRON_NODE_VAL_DECL** (INSTR-04 — immutable binding):
- Cast to `Iron_ValDecl*` (or `Iron_VarDecl*` — check ast.h for the actual struct used for val declarations)
- `lower_expr(ctx, vd->init)` -> init_val
- `shput(ctx->val_binding_map, vd->name, init_val)` — NO alloca, direct SSA value
- Val bindings do NOT get alloca treatment (per research Pitfall 2)

**IRON_NODE_VAR_DECL** (INSTR-04 — mutable binding):
- Cast to `Iron_VarDecl*`
- Emit alloca in the ENTRY block (blocks[0]), NOT in current_block (per research Pitfall 1):
  ```c
  IronIR_Block *entry = ctx->current_func->blocks[0];
  IronIR_Instr *slot = iron_ir_alloca(ctx->current_func, entry,
                                       vd->declared_type, vd->name, vd->base.span);
  shput(ctx->var_alloca_map, vd->name, slot->id);
  ```
- If init expression exists: lower it, then emit store in current_block:
  ```c
  if (vd->init) {
      IronIR_ValueId init_val = lower_expr(ctx, vd->init);
      iron_ir_store(ctx->current_func, ctx->current_block, slot->id, init_val, vd->base.span);
  }
  ```

**IRON_NODE_ASSIGN** (INSTR-04, INSTR-05, INSTR-06):
- Cast to `Iron_Assign*`
- Lower RHS: `lower_expr(ctx, a->value)` -> rhs
- Switch on target kind:
  - IRON_NODE_IDENT: look up var_alloca_map, emit store
  - IRON_NODE_FIELD_ACCESS: lower object, emit set_field
  - IRON_NODE_INDEX: lower object + index, emit set_index

**IRON_NODE_IF** (CTRL-01, CTRL-02):
- Cast to `Iron_IfExpr*` (check ast.h for actual struct name)
- Lower condition: `lower_expr(ctx, if_node->condition)` -> cond_val
- Create blocks: then_block, else_block (if else branch exists), merge_block
- Emit branch: `iron_ir_branch(fn, current_block, cond_val, then_block->id, else_or_merge->id, span)`
- Switch to then_block, lower then body via `lower_block(ctx, if_node->then_body)`
- If then_block is NOT already terminated (no return/break in body): emit jump to merge
- If else branch exists: switch to else_block, lower else body, if not terminated emit jump to merge
- Switch to merge_block — this is the continuation point
- For if-EXPRESSIONS (value-producing): use phi at merge to merge then/else values. Check if the if node is used as an expression (its resolved_type is not void/NULL). If so, the last expression in each branch produces a value; add phi incoming from then_exit and else_exit blocks.

**IRON_NODE_WHILE** (CTRL-01, CTRL-02):
- Create blocks: header, body, exit
- Emit jump from current_block to header
- Switch to header, lower condition, emit branch to body/exit
- Save loop context: `old_exit = ctx->loop_exit_block; old_continue = ctx->loop_continue_block; old_depth = ctx->loop_scope_depth;`
- Set: `ctx->loop_exit_block = exit; ctx->loop_continue_block = header; ctx->loop_scope_depth = ctx->defer_depth;`
- Switch to body, lower body statements
- If body not terminated: emit jump back to header (back-edge)
- Restore loop context
- Switch to exit

**IRON_NODE_FOR** (CTRL-01, CTRL-02):
- For non-parallel for loops: lower as while loop pattern
- Create blocks: pre_header, header, body, increment, exit
- In pre_header: initialize loop variable (alloca in entry + store initial value)
- In header: load counter, compare to bound, branch to body or exit
- In body: lower body statements, jump to increment
- In increment: load counter, add 1, store, jump to header
- Set loop_exit_block = exit, loop_continue_block = increment
- Check ast.h for Iron_ForLoop struct fields (loop variable name, range start, range end, body)

**IRON_NODE_MATCH** (CTRL-04):
- Cast to `Iron_Match*`
- Lower subject: `lower_expr(ctx, match->subject)` -> subject_val
- Extract discriminant (for enum matching): `iron_ir_get_field(fn, block, subject_val, "__tag", int_type, span)` -> tag_val
- Create one block per arm + default block + join block
- Build case_values[] and case_blocks[] arrays for IR_SWITCH
- Emit: `iron_ir_switch(fn, current_block, tag_val, default_block->id, case_values, case_blocks, case_count, span)`
- For each arm:
  - Switch to arm block
  - If arm has binding: cast subject to variant type, get_field for bound variables, record in val_binding_map
  - Lower arm body
  - If not terminated: emit jump to join_block
- For match-EXPRESSIONS: add phi at join_block with incoming from each arm's exit block
- Switch to join_block
- arrfree case_values and case_blocks

**IRON_NODE_RETURN** (CTRL-03, MEM-04):
- Cast to `Iron_Return*`
- CRITICAL: emit defers BEFORE the return terminator (per research anti-pattern):
  ```c
  emit_defers_ir(ctx, ctx->function_scope_depth);
  ```
- If has return value: `lower_expr(ctx, ret->value)` -> ret_val, `iron_ir_return(fn, block, ret_val, false, fn->return_type, span)`
- If void return: `iron_ir_return(fn, block, IRON_IR_VALUE_INVALID, true, NULL, span)`
- After return, the current_block is terminated. Create a new unreachable block to absorb any dead code after the return (prevents "instruction after terminator" verifier error):
  ```c
  IronIR_Block *dead = new_block(ctx, "dead");
  switch_block(ctx, dead);
  ```

**IRON_NODE_DEFER** (MEM-04):
- Cast to `Iron_Defer*`
- Push the deferred expression onto the current scope's defer stack:
  ```c
  arrput(ctx->defer_stacks[ctx->defer_depth - 1], defer_node->body);
  ```
- Do NOT lower the deferred expression now — it gets lowered when emit_defers_ir is called at scope exit

**IRON_NODE_FREE** (MEM-03):
- Cast to `Iron_Free*`
- `lower_expr(ctx, free_node->value)` -> val
- `iron_ir_free(fn, block, val, span)`

**IRON_NODE_LEAK**:
- This is a no-op in IR (leak tells the compiler to skip auto-free, which is already handled by escape analysis flags). Emit nothing, or lower the inner expression and discard the result.

**IRON_NODE_SPAWN** (CONC-02 partial — placeholder for Plan 03):
- Assign lifted name: `char name[64]; snprintf(name, sizeof(name), "__spawn_%d", ctx->module->spawn_counter++);`
- Push LiftPending descriptor for LIFT_SPAWN
- Emit `iron_ir_spawn(fn, block, name, pool_val, handle_name, type, span)` with pool_val from the spawn expression's pool reference
- Check ast.h for Iron_Spawn struct fields

**IRON_NODE_DRAW**:
- Per locked decision: IRON_NODE_DRAW is being removed from the parser in Plan 08-03. For now, emit poison with a diagnostic message "draw keyword deprecated — use raylib.draw()"

**IRON_NODE_BLOCK**:
- Cast to `Iron_Block*`
- Call `lower_block(ctx, block_node)` (which handles push/pop defer scope and stmt iteration)

**Expression-as-statement** (any expression node kind appearing as a statement):
- Call `lower_expr(ctx, node)` and discard the result ValueId. This handles bare function calls used as statements (`foo();`).

**default**:
- If the node kind is an expression kind, call `lower_expr(ctx, node)` (expression-as-statement)
- Otherwise emit poison
