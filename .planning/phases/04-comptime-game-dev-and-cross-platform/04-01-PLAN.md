<objective>
Add extern func declarations and draw {} block support to the full compiler pipeline (lexer, parser, resolver, codegen).

Purpose: These are the foundational language features that enable raylib integration. Without extern func, Iron cannot call C library functions. Without draw {}, the game loop pattern from the spec is incomplete.

Output: Modified compiler that parses `extern func`, resolves extern symbols, emits unmangled C calls with string auto-conversion, and lowers `draw {}` to BeginDrawing/EndDrawing pairs.
</objective>

1. **gen_exprs.c — extern func call emission:** In the `IRON_NODE_CALL` case, BEFORE the existing `iron_mangle_name` path (and after builtin checks), add a check:
   ```c
   if (callee_id->resolved_sym &&
       callee_id->resolved_sym->is_extern) {
       const char *c_name = callee_id->resolved_sym->extern_c_name;
       if (!c_name) c_name = callee_id->name; /* fallback: use Iron name as-is */
       iron_strbuf_appendf(sb, "%s(", c_name);
       for (int i = 0; i < call->arg_count; i++) {
           if (i > 0) iron_strbuf_appendf(sb, ", ");
           /* Auto-convert Iron_String to const char* for extern params.
            * Check if the argument's resolved_type is IRON_TYPE_STRING. */
           Iron_Node *arg = call->args[i];
           bool is_string_arg = false;
           /* Check various expression nodes for resolved_type */
           if (arg->kind == IRON_NODE_STRING_LIT || arg->kind == IRON_NODE_INTERP_STRING) {
               is_string_arg = true;
           } else if (arg->kind == IRON_NODE_IDENT) {
               Iron_Ident *id = (Iron_Ident *)arg;
               if (id->resolved_type && id->resolved_type->kind == IRON_TYPE_STRING)
                   is_string_arg = true;
           }
           if (is_string_arg) {
               /* Wrap: emit iron_string_cstr(&(expr)) */
               iron_strbuf_appendf(sb, "iron_string_cstr(&(Iron_String){");
               /* Need to emit the string value inline for conversion */
               iron_strbuf_appendf(sb, "iron_string_from_literal(");
               /* Actually, simpler approach: emit expression then extract cstr */
               /* Reset and use: */
           }
           /* Simplified correct approach: */
           /* For string literals, emit just the raw C string */
           if (arg->kind == IRON_NODE_STRING_LIT) {
               Iron_StringLit *slit = (Iron_StringLit *)arg;
               iron_strbuf_appendf(sb, "\"%s\"", slit->value);
           } else {
               emit_expr(sb, arg, ctx);
           }
       }
       iron_strbuf_appendf(sb, ")");
       break;
   }
   ```
   **Key behavior:** When `is_extern` is true on the resolved symbol:
   - Use `extern_c_name` as the function name (no `Iron_` prefix)
   - For string literal arguments (IRON_NODE_STRING_LIT), emit as raw C strings `"text"` instead of `iron_string_from_literal(...)`, because C functions like raylib's `InitWindow` expect `const char*`
   - For identifier arguments whose resolved_type is IRON_TYPE_STRING, wrap with `iron_string_cstr(&(` ... `))` to extract the C string from Iron_String
