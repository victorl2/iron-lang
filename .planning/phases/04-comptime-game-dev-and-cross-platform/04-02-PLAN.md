     ```

     typedef struct {

         Iron_DiagList *diags;
         Iron_Scope    *global_scope;
         int            steps;
         int            step_limit;     /* 1,000,000 default */

     ```

     - `Iron_Node *iron_comptime_val_to_ast(Iron_ComptimeVal *val, Iron_Arena *arena, Iron_Span span);`

2. **Create src/comptime/comptime.c** implementing the tree-walking interpreter:

   **iron_comptime_eval_expr(ctx, node)** — main dispatch:
   - `IRON_NODE_INT_LIT`: return CVAL_INT with parsed value
   - `IRON_NODE_FLOAT_LIT`: return CVAL_FLOAT with parsed value
   - `IRON_NODE_BOOL_LIT`: return CVAL_BOOL
   - `IRON_NODE_STRING_LIT`: return CVAL_STRING
   - `IRON_NODE_BINARY`: evaluate left and right, apply operator (+, -, *, /, %, ==, !=, <, >, <=, >=, and, or). For Int operands, do int64_t arithmetic. For Float, do double. Division by zero emits compile error with call trace.
   - `IRON_NODE_UNARY`: evaluate operand, apply - or !
   - `IRON_NODE_CALL`: Look up callee function in global_scope. If found and it is a FuncDecl with a body, push call onto call_stack, increment steps, create local scope with param bindings, evaluate body statements. Return the value from the return statement.
   - `IRON_NODE_IDENT`: Look up variable in local scope (comptime has its own local scope stack for function evaluation).
   - `IRON_NODE_ARRAY_LIT`: Evaluate each element, return CVAL_ARRAY.
   - `IRON_NODE_CONSTRUCT`: Evaluate each arg, return CVAL_STRUCT.
   - `IRON_NODE_HEAP`, `IRON_NODE_RC`: Emit compile error "heap/rc not allowed in comptime context" (CT-03).
   - `IRON_NODE_COMPTIME`: Evaluate the inner expression (nested comptime is valid but redundant).

   **Statement evaluation** for function bodies:
   - `IRON_NODE_BLOCK`: evaluate each statement
   - `IRON_NODE_VAL_DECL`/`IRON_NODE_VAR_DECL`: evaluate init, bind in local scope
   - `IRON_NODE_ASSIGN`: evaluate value, update local binding
   - `IRON_NODE_RETURN`: evaluate value, signal return via a special return value flag or longjmp-like mechanism (simplest: return the value and check a `had_return` flag)
   - `IRON_NODE_IF`: evaluate condition, execute matching branch
   - `IRON_NODE_WHILE`: evaluate condition in loop, execute body, check step limit each iteration
   - `IRON_NODE_FOR`: evaluate iterable (must be an int for range), loop from 0 to N, bind loop var, execute body, check step limit each iteration

   **Step counting:** Increment `ctx->steps` at every function call entry and every loop iteration. If `ctx->steps > ctx->step_limit`, emit error:
   ```
   "comptime evaluation exceeded step limit (1000000 steps)"
   ```
   Include call trace in the diagnostic hint.

   **iron_comptime_val_to_ast(val, arena, span):** Convert the computed value back to an AST literal node:
   - CVAL_INT -> IRON_NODE_INT_LIT (value as string, e.g. "55")
   - CVAL_FLOAT -> IRON_NODE_FLOAT_LIT (value as string with enough precision)
   - CVAL_BOOL -> IRON_NODE_BOOL_LIT
   - CVAL_STRING -> IRON_NODE_STRING_LIT
   - CVAL_ARRAY -> IRON_NODE_ARRAY_LIT with element nodes
   CRITICAL: Set resolved_type on the new literal nodes (copy from the original COMPTIME node's resolved_type) so codegen type mapping works.

   **iron_comptime_apply(program, ...):** Walk all declarations. For each function body, recursively find IRON_NODE_COMPTIME nodes. For each found:
   1. Evaluate inner expression via iron_comptime_eval_expr
   2. Convert result to AST literal via iron_comptime_val_to_ast
   3. REPLACE the IRON_NODE_COMPTIME node in-place (overwrite the parent's pointer to this node with the new literal node)
   This uses a visitor pattern or manual AST walk. The key is finding parent pointers to do the replacement.

3. **diagnostics.h:** Add two new error codes:
   - `IRON_ERR_COMPTIME_STEP_LIMIT` (in the error code enum, e.g., value 230)
   - `IRON_ERR_COMPTIME_RESTRICTION` (e.g., value 231)
   - `IRON_ERR_COMPTIME_ERROR` (e.g., value 232) for division by zero, etc.
