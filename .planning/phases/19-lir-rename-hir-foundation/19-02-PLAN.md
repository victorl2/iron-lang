<objective>
Build the HIR data structures, enums, constructors, and unit tests that represent all Iron language constructs at a high level.

Purpose: HIR sits between AST and LIR in the compilation pipeline. It uses structured control flow (tree nodes, not CFG), named variables (VarId + name_table), and language-level constructs (closures, spawn, defer). This plan creates the complete data layer that the printer, verifier, and eventually the lowering passes (Phase 20) will operate on.

Output: src/hir/hir.h (all types), src/hir/hir.c (constructors), tests/hir/test_hir_data.c (unit tests proving all node kinds can be constructed and composed).
</objective>

    7. IronHIR_Expr tagged union: kind + span + type + union of per-kind structs (int_lit, float_lit, string_lit, interp_string, bool_lit, ident, binop, unop, call, method_call, field_access, index, slice, closure, heap, rc, construct, array_lit, await_expr, cast, is_null, is_not_null, func_ref, parallel_for, comptime, is_check)

    8. IronHIR_Func and IronHIR_Module structs. Module owns arena and name_table.

    9. All constructor declarations.

    Use the BinOp/UnOp enum from ast.h if one exists; otherwise define IronHIR_BinOp and IronHIR_UnOp enums mirroring the AST operators (+, -, *, /, %, ==, !=, <, <=, >, >=, &&, ||, !, -neg).

    **hir.c** -- Implement all constructors following the LIR alloc_instr pattern:
    - iron_hir_module_create(name): creates arena (64KB initial), zeroed module, next_var_id=1, name_table[0] = sentinel
    - iron_hir_module_destroy(mod): arrfree funcs, arrfree name_table, destroy arena
    - iron_hir_alloc_var(mod, name, type): assigns next_var_id++, arrput to name_table
    - iron_hir_var_name(mod, id): returns name_table[id].name
    - iron_hir_func_create: ARENA_ALLOC + memset + field assignment
    - iron_hir_module_add_func: arrput + update func_count
    - iron_hir_block_create + iron_hir_block_add_stmt: arrput pattern
    - One constructor per StmtKind (13 functions)
    - One constructor per ExprKind (24+ functions)
    Each constructor: ARENA_ALLOC, memset to 0, set kind, set span, set per-kind fields, return pointer.

    **CMakeLists.txt** -- Add `src/hir/hir.c` to the iron_compiler source list (after the lir sources).
