    **3. Create src/hir/hir_print.c:**

    Include hir/hir.h, util/strbuf.h, analyzer/types.h, and util/arena.h.

    Implement `iron_hir_print(const IronHIR_Module *module)`:
    - Create Iron_StrBuf (initial capacity 4096)
    - Create a temp arena for iron_type_to_string calls
    - For each func in module->funcs: call print_func(sb, func, mod, 0, tmp)
    - Return strdup of iron_strbuf_get, then free strbuf and temp arena

    Static helper `indent(Iron_StrBuf *sb, int depth)`: emit depth*2 spaces.

    Static helper `print_func(sb, func, mod, depth, tmp)`:
    - Print `FuncDecl(name) -> return_type\n`
    - For each param: print `  Param(name: type)\n` at depth+1
    - Call print_block(sb, func->body, mod, depth+1, tmp)

    Static helper `print_block(sb, block, mod, depth, tmp)`:
    - Print `Block\n` at current depth
    - For each stmt: call print_stmt(sb, stmt, mod, depth+1, tmp)

    Static helper `print_stmt(sb, stmt, mod, depth, tmp)`:
    - switch on stmt->kind, each case prints label then recursively prints children at depth+1
    - IRON_HIR_STMT_LET: `LetStmt(name: type)\n` (or `VarStmt(name: type)\n` if mutable), then print init expr at depth+1
    - IRON_HIR_STMT_ASSIGN: `AssignStmt\n`, print target and value at depth+1
    - IRON_HIR_STMT_IF: `IfStmt\n`, print condition at depth+1 prefixed with `Cond\n`, print then_body, optionally `Else\n` + else_body
    - IRON_HIR_STMT_WHILE: `WhileStmt\n`, condition + body
    - IRON_HIR_STMT_FOR: `ForStmt(var_name)\n`, iterable + body
    - IRON_HIR_STMT_MATCH: `MatchStmt\n`, scrutinee + arms
    - IRON_HIR_STMT_RETURN: `ReturnStmt\n`, optional value
    - IRON_HIR_STMT_DEFER: `DeferStmt\n`, body stmt
    - IRON_HIR_STMT_BLOCK: delegate to print_block
    - IRON_HIR_STMT_EXPR: `ExprStmt\n`, print expr
    - IRON_HIR_STMT_FREE: `FreeStmt\n`, print value
    - IRON_HIR_STMT_SPAWN: `SpawnStmt\n`, print body
    - IRON_HIR_STMT_LEAK: `LeakStmt\n`, print value

    Static helper `print_expr(sb, expr, mod, depth, tmp)`:
    - switch on expr->kind, each case prints label and recursively prints children
    - IRON_HIR_EXPR_INT_LIT: `Literal(%lld)\n`
    - IRON_HIR_EXPR_FLOAT_LIT: `Literal(%g)\n`
    - IRON_HIR_EXPR_STRING_LIT: `Literal("%s")\n`
    - IRON_HIR_EXPR_BOOL_LIT: `Literal(true/false)\n`
    - IRON_HIR_EXPR_NULL_LIT: `Literal(null)\n`
    - IRON_HIR_EXPR_IDENT: `Ident(var_name)\n`
    - IRON_HIR_EXPR_BINOP: `BinOp(op_str)\n`, print left/right at depth+1
    - IRON_HIR_EXPR_UNOP: `UnOp(op_str)\n`, print operand
    - IRON_HIR_EXPR_CALL: `Call\n`, print callee + args
    - IRON_HIR_EXPR_METHOD_CALL: `MethodCall(method_name)\n`, print object + args
    - IRON_HIR_EXPR_FIELD_ACCESS: `FieldAccess(field)\n`, print object
    - IRON_HIR_EXPR_INDEX: `Index\n`, print object + index
    - IRON_HIR_EXPR_CLOSURE: `Closure\n`, print params + body
    - IRON_HIR_EXPR_PARALLEL_FOR: `ParallelFor(var)\n`, range + body
    - All other kinds: appropriate label + children

    Use 2-space indentation per depth level per locked decision.

    **4. Create src/hir/hir_verify.c:**

    Include hir/hir.h, diagnostics/diagnostics.h, util/arena.h, vendor/stb_ds.h.

    Implement `iron_hir_verify(const IronHIR_Module *module, Iron_DiagList *diags, Iron_Arena *arena)`:
    - Record errors_before = diags->error_count
    - For each func in module->funcs: call verify_func(func, mod, diags, arena)
    - Return diags->error_count == errors_before

    **Scope tracking:** Use a scope stack implemented as a dynamic array of scope frames. Each frame is a stb_ds int-keyed hash map (hmput/hmget by VarId) tracking which variables are visible.
    - push_scope(): arrput a new empty hash map
    - pop_scope(): hmfree the top map, then arrdel (or stbds_header adjust arrlen)
    - declare_var(scope_stack, var_id, span, diags, arena): check top scope for duplicate, then hmput
    - lookup_var(scope_stack, var_id): search from top to bottom of stack

    Static helper `verify_func(func, mod, diags, arena)`:
    - Validate func->name != NULL, func->body != NULL
    - Push a scope, declare all parameters
    - Call verify_block(func->body, mod, scope_stack, diags, arena)
    - Pop scope

    Static helper `verify_block(block, mod, scope_stack, diags, arena)`:
    - Push a new scope frame
    - For each stmt: call verify_stmt
    - Pop scope

    Static helper `verify_stmt(stmt, mod, scope_stack, diags, arena)`:
    - Validate stmt != NULL (emit IRON_ERR_HIR_NULL_POINTER if null)
    - switch on stmt->kind:
      - IRON_HIR_STMT_LET: declare var in scope, verify init expr if present
      - IRON_HIR_STMT_ASSIGN: verify target + value exprs
      - IRON_HIR_STMT_IF: verify condition, verify then_body block (new scope), verify else_body if present
      - IRON_HIR_STMT_WHILE: verify condition, verify body
      - IRON_HIR_STMT_FOR: push scope, declare loop var, verify iterable, verify body, pop scope
      - IRON_HIR_STMT_MATCH: verify scrutinee, verify each arm (push scope per arm for pattern bindings)
      - IRON_HIR_STMT_RETURN: verify value if present
      - IRON_HIR_STMT_DEFER: verify body stmt
      - IRON_HIR_STMT_BLOCK: verify nested block
      - IRON_HIR_STMT_EXPR: verify expr
      - IRON_HIR_STMT_FREE: verify value
      - IRON_HIR_STMT_SPAWN: verify body block
      - IRON_HIR_STMT_LEAK: verify value

    Static helper `verify_expr(expr, mod, scope_stack, diags, arena)`:
    - Validate expr != NULL
    - switch on expr->kind:
      - IRON_HIR_EXPR_IDENT: lookup_var -- emit IRON_ERR_HIR_USE_BEFORE_DEF if not found
      - IRON_HIR_EXPR_BINOP: verify left + right. If both have non-NULL types, check compatibility (emit IRON_ERR_HIR_TYPE_MISMATCH if incompatible)
      - IRON_HIR_EXPR_CALL: verify callee, verify each arg
      - IRON_HIR_EXPR_METHOD_CALL: verify object, verify each arg
      - IRON_HIR_EXPR_FIELD_ACCESS: verify object
      - IRON_HIR_EXPR_INDEX: verify object + index
      - IRON_HIR_EXPR_CLOSURE: push scope, declare closure params, verify body, pop scope
      - IRON_HIR_EXPR_PARALLEL_FOR: push scope, declare loop var, verify range + body, pop scope
      - Other exprs: verify child exprs recursively
    - Structural checks: any required child pointer that is NULL emits IRON_ERR_HIR_STRUCTURAL
