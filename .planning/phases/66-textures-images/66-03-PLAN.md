```c
/* Generic Image* mutator template (TEX-05 + TEX-07).
 * Iron func: image.<op>(img: Image, ...args) -> Image {} */
struct Iron_Image Iron_image_<op>(struct Iron_Image img /*, args */) {
    Image src;
    memcpy(&src, &img, sizeof(Image));
    /* memcpy any struct args (Color/Rectangle/Vector2/Image) here */
    Image<Op>(&src /*, args */);   /* raylib mutates src in place */
    struct Iron_Image out;
    memcpy(&out, &src, sizeof(struct Iron_Image));
    return out;
}
```

- No-arg mutators (alpha_premultiply, mipmaps, flip_vertical, flip_horizontal, rotate_cw, rotate_ccw, color_invert, color_grayscale): memcpy img in, call `ImageFoo(&src)`, memcpy out
- Scalar-arg (int/float) mutators (alpha_crop, blur_gaussian, resize, resize_nn, rotate, color_contrast, color_brightness, format): memcpy img in, call `ImageFoo(&src, (int)arg)` or `(&src, arg_float)`, memcpy out

- Rectangle-arg mutators (crop): memcpy img + memcpy Rectangle
- Double-Color mutators (alpha_clear has Color+Float, color_replace has Color+Color): memcpy both Colors as needed
- Image-arg mutator (alpha_mask): memcpy img + memcpy mask Image, call `ImageAlphaMask(&src, mask)`, memcpy out
- Multi-int mutators (dither, resize_canvas): cast each int32_t arg to `(int)` at raylib call site
