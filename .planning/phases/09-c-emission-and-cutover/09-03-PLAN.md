Fix two bugs in the C emission pipeline:

**Bug 1: FUNC_REF mangles extern function names (extern_basic failure)**

In `src/ir/emit_c.c`, the `IRON_IR_FUNC_REF` case (around line 1139-1145) blindly calls `mangle_func_name()` on all function names. This produces `Iron_puts` for `extern func puts`, but extern functions must use their raw `extern_c_name`.

Fix the `IRON_IR_FUNC_REF` case to look up the IrFunc by name in the module and check `is_extern`:

```c
case IRON_IR_FUNC_REF: {
    const char *raw_name = instr->func_ref.func_name;
    const char *c_name = NULL;

    /* Check if this references an extern function */
    for (int fi = 0; fi < ctx->module->func_count; fi++) {
        IronIR_Func *f = ctx->module->funcs[fi];
        if (strcmp(f->name, raw_name) == 0 && f->is_extern) {
            c_name = f->extern_c_name ? f->extern_c_name : raw_name;
            break;
        }
    }
    if (!c_name) {
        c_name = mangle_func_name(raw_name, ctx->arena);
    }

    emit_indent(sb, ind);
    iron_strbuf_appendf(sb, "void* ");
    emit_val(sb, instr->id);
    iron_strbuf_appendf(sb, " = (void*)%s;\n", c_name);
    break;
}
```

Also apply the same extern check in the CALL instruction's direct-call path (around line 650-666) where `mangle_func_name` is called for `fd->name`. When `fd->is_extern` is true, use `fd->extern_c_name` instead.

And in the indirect call FUNC_REF detection path (around line 670-679), when the value_table lookup finds a FUNC_REF, apply the same extern check:

```c
if (fn->value_table[fptr]->kind == IRON_IR_FUNC_REF) {
    const char *ref_name = fn->value_table[fptr]->func_ref.func_name;
    const char *c_name = NULL;
    /* Check extern */
    for (int fi = 0; fi < ctx->module->func_count; fi++) {
        IronIR_Func *f = ctx->module->funcs[fi];
        if (strcmp(f->name, ref_name) == 0 && f->is_extern) {
            c_name = f->extern_c_name ? f->extern_c_name : ref_name;
            break;
        }
    }
    if (!c_name) {
        c_name = mangle_func_name(ref_name, ctx->arena);
    }
    iron_strbuf_appendf(sb, "%s(", c_name);
    emitted_direct = true;
}
```

Consider extracting a helper `resolve_func_c_name(EmitCtx *ctx, const char *ir_name)` that encapsulates the extern check + mangle logic to avoid code duplication.
