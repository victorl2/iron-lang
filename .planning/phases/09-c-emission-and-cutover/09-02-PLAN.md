IMPORTANT: The IR print must happen BEFORE `iron_ir_emit_c()` is called, because emit_c modifies the IR in-place for phi elimination. So restructure the code flow:

4. If verbose: print generated C
5. Continue with write_temp_c / invoke_clang

The verbose IR output format:
```c
if (opts.verbose && ir_module) {

        free(ir_text);
    }
}
```

After emit_c:
