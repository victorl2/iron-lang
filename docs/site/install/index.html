Install Iron

It looks like you're running a Unix-like OS. To install Iron, run the following in your terminal:

curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/victorl2/iron-lang/main/scripts/install.sh | sh

This downloads and runs install.sh, which installs the iron binary to ~/.iron/bin and configures your shell PATH automatically.

It looks like you're running Windows. Iron currently supports macOS and Linux.

You can install Iron using WSL (Windows Subsystem for Linux). Open a WSL terminal and run:

curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/victorl2/iron-lang/main/scripts/install.sh | sh

Native Windows support is planned. See the GitHub repo for updates.
